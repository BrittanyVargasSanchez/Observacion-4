# Descripción del sitio
Este sitio es un sitio de prueba de una tienda en línea llamada Geek Store. Esta vende una breve variedad de productos tecnológicos y tiene su propio sistema de sesión.

# Recomendaciones
Hay varios lugares de mejora con este sitio, cosas como el diseño, la funcionalidad, la cantidad de productos y el registro. 
- El diseño es muy simple, con varios problemas notables como el logo no teniendo fondo vacio, el logo y la barra de buscador estando pegadas y la falta de destinción entre el cuerpo y el pie de página
- Tiene buena funcionalidad, teniendo hasta un sistema de sesión, pero hay varios problemas pequeños como el pie de página llevando al index en vez de otra página.
- La cantidad de productos podría ser mayor, ya que hay categorias con solo un producto. 
- No hay parámetros en el registro, uno puede registrarse con un email que no es posible (ej: 1@1.1), colocar un teléfono inexistente (ej: 1), y permite colocar una contraseña de un carácter.
